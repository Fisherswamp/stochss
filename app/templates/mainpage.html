Welcome to StochSS Stochastic Simulation Service

Welcome to the StochSS App v1.0! In this first version, we provide a Model Editor to create new well-mixed stochastic models, or to import existing StochKit2 models expressed in its native XML format. Once you have defined your model, we let you run simulations with StochKit2.



Want to run simulations locally?

To use StochSS to run StochKit simulations on your local machine, you need to install StochKit2.0.7. Download StochKit from Sourceforge and follow the install instructions for your operating system. StochSS treats StochKit as a computational plugin, so once it is installed, you need to tell us where it is located on your computer. Note that it is not necessary to install StochKit if you only plan to use Cloud resources for your simulations.


Want to run simulations in the Cloud?

StochSS will let you launch computational nodes in the Amazon EC2 compute cloud and use them to run your simulations, all with the click of a button! To use this feature, you need to sign up for an account with Amazon Web Services and then enter your AWS credentials.


For more information, consult the User Manual. Also, make sure to join the StochSS Users Google group to recieve information about updates and to ask questions.
